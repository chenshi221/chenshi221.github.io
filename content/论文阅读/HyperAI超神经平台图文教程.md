## 3. 注册与上手流程

### Step 1：注册账号

访问 https://app.hyper.ai/console ，注册 HyperAI 账号。

### Step 2：了解平台资源

注册后可以看到平台的完整功能：
- **教程**：数百个 Jupyter Notebook，涵盖从入门到前沿
- **数据集**：多模态数据集资源
- **资讯**：每日 AI 行业动态
- **论文**：前沿研究论文
- **SOTA/LMM/GPU 排行榜**：基准测试工具

### Step 3：创建算力容器

在控制台左侧「模型训练」处，点击「+」创建新容器：

1. **选择算力** — RTX 5090 / RTX PRO 6000 / H100
2. **选择计费方式** — 按需付费 / 按天 / 按周 / 按月
3. **选择镜像** — 内置多个版本（PyTorch、TensorFlow 等）
4. **选择接入方式** — Jupyter 工作空间 / 脚本上传
5. **输入容器名称**
6. **绑定数据仓库**（可选）
7. **设置空闲关闭时间**
8. **配置端口映射**（可选）
9. **点击「执行」**

MEDIA:/Users/chenshi/.hermes/cache/screenshots/browser_screenshot_d1c375d3e3cb48bcb2a89a221502fbb4.png

### Step 4：开始使用

容器状态变为「运行中」后，点击「打开工作空间」即可进入 JupyterLab 环境。

**重要：数据存储说明**
- `/`（系统盘）：临时存储，容器关闭后**数据会丢失**
- `/hyperai/home`（工作空间）：**持久化存储**，数据会自动保存

> ⚠️ 请始终将代码、数据、模型等文件放在 `/hyperai/home` 目录下！

## 5. 计费与资源管理

### 计费方式
- **按需付费（Pay As You Go）**：用多少扣多少
- **按天/按周/按月**：周期内自由使用，到期自动续费

### 注意事项
- 容器状态为「运行中」时才开始计费
- 按需付费余额耗尽时，容器会被强制停止
- 不活跃项目（1 个月无操作）会被自动清理
- 存储空间不足时可购买扩展，支持自动续费

### 查看用量
- 控制台 → 设置 → 配额用量 → 概览
- 可查看计算和存储的详细使用记录

## 7. 优缺点总结

### ✅ 优点
- **免费 CPU**：永久免费，适合学习
- **性价比高**：RTX 5090 仅 $0.35/h，Pro 会员 $4.99/月含 30h GPU
- **教程丰富**：数百个 Jupyter Notebook，覆盖全面
- **一站式**：算力 + 教程 + 论文 + 数据集 + 排行榜
- **国内平台**：中文支持好，无需翻墙

### ❌ 缺点
- GPU 型号选择相对有限（目前 3 种）
- 免费 CPU 配置较低（8 vCPU / 16 GB）
- 不活跃项目会被自动清理
- 存储空间基础配额较小（20 GB）
