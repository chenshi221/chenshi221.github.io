# ImgEdit: A Unified Image Editing Dataset and Benchmark

北京大学深圳 + 鹏城实验室 + Rabbitpre AI，2025。

## 核心结论

- 提出 **ImgEdit**，统一图像编辑数据集与 benchmark。
- 覆盖多种编辑类型和图像域。
- 提供标准化的训练和评估协议。

## 在图像编辑数据集生态中的位置

| 数据集 | 规模 | 特点 |
|--------|------|------|
| InstructPix2Pix | 450K | 合成数据，奠基 |
| MagicBrush | ~10K | 人工标注，高质量 |
| UltraEdit | ~4M | 真实图像，LLM in-context |
| AnyEdit | 2.5M | 20+ 类型，task-aware |
| ImgEdit | 中大规模 | 统一 benchmark |
| OpenGPT-4o-Image | 80K | 层级分类，GPT-4o 生成 |
